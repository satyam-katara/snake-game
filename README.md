# 🐍 Snake Game

## 📌 Overview
A classic Snake Game built using Python and the Turtle Graphics library. Control the snake, eat food to grow longer, and avoid colliding with the walls or your own body to achieve the highest score.

## ✨ Features
- Interactive snake movement using keyboard controls
- Random food generation
- Snake grows after eating food
- Real-time score tracking
- Collision detection with walls
- Collision detection with the snake's own body
- Game Over screen

## 🛠️ Technologies Used
- Python 3
- Turtle Graphics

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/satyam-katara/snake-game.git
```

2. Navigate to the project folder:

```bash
cd snake-game
```

3. Run the game:

```bash
python main.py
```
